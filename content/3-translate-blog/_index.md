### [Blog 1 - Bảo mật Amazon Elastic VMware Service (Amazon EVS) với AWS Network Firewall](./3.1)

Bài viết này trình bày cách bảo vệ môi trường Amazon EVS bằng AWS Network Firewall. Giải pháp tập trung kiểm tra lưu lượng giữa EVS, các VPC, hệ thống on-premises và internet thông qua một điểm kiểm soát duy nhất. Kiến trúc sử dụng AWS Transit Gateway kết hợp tích hợp native với AWS Network Firewall để quản lý chính sách, log và giám sát tập trung.
